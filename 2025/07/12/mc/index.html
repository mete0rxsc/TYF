MC联机教程

发表于2025-07-12|更新于2025-07-12
|浏览量:

！由于我们现在大部分联机已经开始使用正版，盗版用户请看下文！

正版用户直接跳转至正版板块

盗版用户购买正版板块

  1. 登录官网：https://www.minecraft.net/zh-hans
    在此处他会弹窗提示你跳转到网易版本，请选择下面的继续浏览该网站
  2. 登录账号
    点击右上角的账户，选择登录或创建账户
    在此处他会弹窗提示你跳转到网易版本，请选择下面的继续浏览该网站
    使用微软账号登录，如果你没有，请自行注册
    登陆完后即可
  3. 打开电脑自带的Microsoft Store
    一般他会自己出现在你的开始菜单中

    搜索 Minecraft
    选择第一个 Minecraft: Java & Bedrock Editionfor PC
  4. 购买游戏

如果你有正版后，请看这里

  1. 下载PCL2启动器 点我进入下载页面
  2. 登录你的正版账号
  3. 下载对应整合包
  4. 进行多人游戏，域名多为gm.xscnet.cn
